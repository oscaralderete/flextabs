Flextab - Responsive tabs system based on flexbox and handled by JS

Vue is my fav JS framework, so you can integrate it with Flextab easily!

Tabs let you put a lot of information/processes in one place so Flextab could be a usefool tool in your projects, specially SPAs.

(Check scripts.js file to check how to implement Flextab, it's really easy.)

    const thirdTab = new Flextab({
	id: 'basic-sample',
	active_index: 2,
	callback: function(index){
		if(index===0){
			console.log('first tab was clicked...');
			vue2.populatePeople();
		}
	}
});

    As you can see above, these tabs were set to start showing third tab content because third tab index is 2. Programmatic navigation is available every where on markup.

    You can pass callback functions too, in this example we're fetching remote data every time we press on "Lorem" tab.
